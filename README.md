# 🌍 World Happiness Data Analysis

## 📌 Project Overview

This project analyzes the World Happiness dataset to understand the factors influencing happiness scores across different countries. The objective is to explore relationships between economic, social, and health indicators and overall happiness levels.

## 📂 Dataset

* Source: World Happiness Report Dataset (Kaggle)
* Contains country-wise data including GDP, life expectancy, social support, freedom, and happiness score

## ⚙️ Approach

* Loaded dataset using Pandas
* Cleaned missing and inconsistent values
* Selected relevant features for analysis

## 📊 Exploratory Data Analysis

* Analyzed distribution of happiness scores
* Compared happiness levels across countries
* Studied relationships between GDP, life expectancy, and happiness
* Visualized correlations using heatmaps and scatter plots

## 🔍 Key Insights

* Higher GDP per capita is strongly associated with higher happiness scores
* Social support plays a significant role in overall happiness
* Countries with better health (life expectancy) tend to be happier
* Strong correlations observed between multiple socio-economic factors
